Question 3:

As a web engineer, describe three performance concerns you need to deal with and how you address them.

Functionality - the logic behind what makes your web application appealing to users needs to operate properly in order to deliver the best product. It'd be a good idea to write tests to validate operations to see if they carry through properly.

Speed - performance testing is important because it allows us to see how efficiently our code is operating. We need to see how our backend is corresponding with our frontend network-wise. We also need to ensure that the HTTP-roundtrips aren't excessive.

Appearane - it is important that our application behaves and looks the way we want it to. As the user navigates through and experiences our product, we need to ensure that proper sizing and placement and such are on par with what we expected it to be.
